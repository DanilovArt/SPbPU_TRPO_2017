Задание 2.
=========================================================================

Требуется обеспечить автоматическую сборку контейнера средствами Gradle.
Результатом работы будет gradlew файл, который собирает Docker контейнер с приложением
(т.е. контейнер должен содержать только исполняемый файл и среду исполнения).

Можно использовать любые плагины для Gradle либо писать самостоятельно на Groovy.

Порядок работы:
1. Дождаться появления issue на выполнение этой задачи в своём репозитории.
2. Выполнить решение в отдельной ветке, согласно git (в т.ч. провести обновление документации)
3. Объединить ветку с решением и основную ветку для репозитория.

Для третьего шага желательно получить аппрув от кого-то из Ваших коллег.
Для этого нужно в настройках проекта добавить их в Collaborators.
